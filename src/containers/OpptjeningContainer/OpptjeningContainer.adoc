=== OpptjeningContainer
Wrapperkomponent for 책 laste data til et xref:_opptjeningview[OpptjeningView]. Komponenten dispatcher en redux action for 책 hente
data fra backend.

[source, javascript]
----
useEffect(() => {
    dispatch(fetchOpptjeningStarted());
});
----

| 200 | Data settes i state og applikasjonen rendres
